# ml-weekend-gap-prediction

Hey guys — quick temporary setup notes for running this locally.

### Install requirements

From the project root, run:

```bash
python3 -m install -r requirements.txt

```
### To build the data
If you want to make any changes and then want to run it such that it
builds.

1. Navigate to
```bash
   /ml-weekend-gap-prediction
```

2. run feature development script
```bash
    ./run_script.sh
```
3. Follow the prompt to select a script to run
4. To exit run just press CTRL + C
